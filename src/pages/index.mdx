# Triangle Web

Высокопроизводительный HTTP-фреймворк, разработанный на основе [Localzet Server](https://github.com/localzet/Server). Призван заменить традиционную архитектуру PHP-FPM. Triangle можно использовать для разработки веб-сайтов, HTTP-интерфейсов или микросервисов. {{ className: 'lead' }}
